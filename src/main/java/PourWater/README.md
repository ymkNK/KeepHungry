# 倒水问题
有两个杯子,容量为A和B,还有个足够大的缸子,要用这俩杯子倒出C的水,判断能不能倒出来

# 原理
能倒的水 x1A+x2B+x3(A+B)+x4(A-B)=C
化简之后 xA+yB=C
#### 定理
欧几米德扩展:
存在xA+yB=g(A,B) g是A,B的最大公约数


#### 结论
如果C能被g整除,那么就是一定能倒出来的

#### 辗转相除法
用较大数除以较小数，再用出现的余数（第一余数）去除除数，再用出现的余数（第二余数）去除第一余数，如此反复，直到最后余数是0
